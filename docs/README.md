# Documentação

Esta pasta contém a documentação detalhada do projeto Dungeon Keeper.

> **Foco:** Inicialmente voltado para D&D 5e e uso presencial, o Dungeon Keeper busca ser um auxílio prático para jogadores e mestres, reforçando a imersão e a socialização nas mesas físicas de RPG.

## Estrutura

- `systems/` - Documentação dos sistemas principais
  - `character/` - Sistema de personagens
  - `combat/` - Sistema de combate
  - `inventory/` - Sistema de inventário
  - `magic/` - Sistema de magias
